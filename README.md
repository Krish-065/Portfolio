# Krish Shah | IT Engineering & Data Science Portfolio

Welcome to my personal developer and research portfolio! I am **Krish Shah**, currently pursuing my **B.Tech in Information Technology** at **Charusat University (Anand, Gujarat)**. My engineering focus is centered on **Data Science**, **Machine Learning**, and building high-performance, interactive fullstack web systems.

This repository hosts a custom, highly interactive React & Vite portfolio built with glassmorphic cards, physics-based canvas animations, and real-time training simulators.

---

## ⚡ Core Interactive Features

*   **Interactive ML Simulator Lab**: Adjust models (Deep Neural Networks, Transformers, Simple Regression), learning rates, epochs, and dataset size. Watch the decision boundaries dynamically fit coordinates on an interactive canvas in real-time as training converges.
*   **Neural Canvas Particle Backdrop**: A responsive, physics-based backdrop rendering node connections that shift and interact dynamically with the cursor.
*   **Developer JSON Terminal Card**: A simulated console card presenting my professional skills profile, education parameters, and aims.
